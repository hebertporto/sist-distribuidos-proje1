# Instalar o NodeJS

## instalar o nodemon:  $ npm install -g nodemon

- Entrar na pasta CLIENT 
  no terminal : $ npm install 

- Abrir outro terminar e entrar na pasta SERVER
  no terminal : $ npm install 

### Dentro de cada terminal:
  $ nodemon index.js 

  o prompt será interativo e mostrá os consoles.
